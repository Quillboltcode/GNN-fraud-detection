# Agent Instructions

## Project Overview
This is a fraud detection project using Graph Neural Networks (GNNs).

## Technology Stack
- Python
- PyTorch Geometric
- NetworkX

## Key Commands
- Install dependencies: `uv sync`
- Run training: `python src/train.py`
- Run tests: `pytest`

## Coding Conventions
- Follow PEP 8
- Use type hints
- Add docstrings to functions

## Linting & Type Checking
- Run linting: `ruff check .`
- Run type checking: `pyright .`
